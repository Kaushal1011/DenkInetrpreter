# DENK Language

For examples of code refer README or test files

## Feature

* Data Types
    * integer
    * real
    * boolean
    * string
* Variables
* Procedures
* Functions
* Flow Control
    * if
    * else if
    * else
* Loops
    * while...do
* Numerical Operators
    * \+
    * \-
    * \*
    * /
    * div
* Logical Operators
    * and
    * or
    * not
* Comparison Operators
    * =
    * <>
    * <
    * <=
    * \>
    * \>=
* Bitwise Operators
    * leftshit(>>)
    * rightshift(<<)
    * not(~)
    * and(&)
    * or(|)
    * xor(^)

The syntax of this language is from inspired from pascal and for more on pascal go to [Pascal Tutorial](http://www.tutorialspoint.com/pascal/pascal_program_structure.htm)
